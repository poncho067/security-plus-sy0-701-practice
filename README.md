# Security+ SY0-701 Practice

Static practice quiz app for CompTIA Security+ SY0-701 study.

## Features

- General mixed quizzes or focused quizzes by exam objective
- Configurable question count and time per question
- Difficulty selector with hard as the default
- Countdown bar per question
- Saved progress in the browser with `localStorage`
- Post-quiz Overview, Retry, Practice more, and Exit flows

## Run Locally

```bash
node server.mjs
```

Then open `http://localhost:4173/`.

The app is static and can also be hosted directly from GitHub Pages.
